# HTML/CSS Project: Student Profile Card

## Project Summary

This project demonstrates how HTML creates structure and CSS creates design. It is simple enough to explain in a 20-30 minute interview demo.

## Concepts Covered

- HTML document structure
- Semantic tags
- CSS selectors
- Flexbox layout
- Responsive design
- Hover effects

## How To Run

Open `index.html` in any browser.

## Teaching Flow

1. Explain that HTML is the skeleton of a webpage.
2. Explain that CSS adds color, spacing, layout, and style.
3. Show the profile card.
4. Change a color or text live.
5. Explain how the design responds on smaller screens.
