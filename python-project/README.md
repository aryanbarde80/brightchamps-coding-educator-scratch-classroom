# Python Project: Number Quest

## Project Summary

Number Quest is a simple guessing game. The computer chooses a secret number, and the player tries to guess it.

## Concepts Covered

- Variables
- Input and output
- Loops
- If/elif/else conditions
- Functions
- Random numbers

## How To Run

```bash
python number_quest.py
```

## Teaching Flow

1. Explain the game rule.
2. Show how Python stores the secret number.
3. Explain the loop that allows repeated guesses.
4. Explain conditions for too high, too low, and correct.
5. Run the program and play one round.
